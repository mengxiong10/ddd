# items 道具系统 PRD

## 问题
武将的武力/智力目前是纯静态属性，玩家没有任何手段去强化武将、或表达对武将的恩威。缺少「装备/赏赐」这一经典策略维度：玩家拿到的道具无处安放，也没有笼络人心（忠诚）的抓手。

## 目标
引入道具与武将忠诚：道具可属于城或武将，能提升武将的有效武力/智力（并经既有公式自动惠及带兵量、开垦/招商）；君主可通过「赏赐」把城中道具给武将（涨忠诚）、通过「没收」收回道具（掉忠诚），从而在不占用武将本月行动的前提下经营人心。

## 关键行为 / 用户故事
1. 作为玩家，我要给某城的道具，赏赐给该城的一名己方武将，让他变强且更忠诚。
2. 作为玩家，我要没收某武将的道具收回城中（以备转赐他人），即便武将会因此掉忠诚。
3. 作为玩家，我希望武将佩戴道具后，其有效武力/智力随之提高，并自动反映到带兵量上限、开垦/招商增量上。
4. 作为玩家，我要能在面板上看到每座城、每名武将各自持有的道具与各道具的加成。
5. 作为玩家，我对武将下达赏赐/没收后，该武将本月仍可被派去执行其他指令（赏赐/没收不占人、立即生效）。

## 关键规则（与用户对齐结论）

### 道具（Item）
- 道具是带唯一 id 的实体，含 `武力加成`、`智力加成`（均为非负整数，可为 0）。
- **归属单一真相源**：在 `GameState` 顶层维护 `items` 集合，每个道具记录其「归属」——属于某城（cityId）**或**属于某武将（officerId），二选一。城/武将各自持有的道具列表一律用 selector 派生，不存第二份（禁止双向冗余）。
- **每名武将最多持 2 件道具**（不变量）。
- 初始道具由 fixture 播种到城。

### 加成（有效属性）
- 武将存「基础武力/基础智力」；派生 **有效武力 = 基础武力 + 所持道具武力加成之和**，**有效智力 = 基础智力 + 所持道具智力加成之和**。
- 所有用到武力/智力的既有公式统一改吃**有效值**：带兵量上限（等级×100 + 武力×10 + 智力×10）、开垦/招商增量。单一真相源、效果涌现，无需在各处手动叠加。

### 武将忠诚（Loyalty）
- 新增武将级属性「忠诚」，范围 `[0, 100]`，普通武将初始 50。
- **君主忠诚派生恒为 100**：忠诚增减只作用于非君主武将；对君主的道具操作照常进行，但其忠诚始终 100。
- 本切片忠诚**仅为状态**（可显示），赏赐 +8、没收 −20 据此增减，**暂无任何下游游戏规则**（叛变/策反等留待后续切片）。

### 赏赐（Reward）指令
- **目标是武将**（非执行人），**不占人、效果时机=下令瞬间**（立即生效），无金/体力成本。
- 作用城 = 目标武将 `cityId`（单一真相源，沿用既有约定）。
- 把一件**属于该城**的道具，转给该城的一名**己方非俘虏**武将；不要求该武将 available（君主命令，武将本月仍可做其他事）。
- 校验失败即 no-op：道具不属于作用城、目标武将已持 2 件、目标为俘虏 等。
- 成功后：道具归属改为该武将；该武将（非君主）忠诚 +8（封顶 100）。

### 没收（Confiscate）指令
- 同样**目标是武将**、**不占人、立即生效**、无金/体力成本。
- 作用城 = 目标武将 `cityId`。
- 把目标武将所持的一件道具收回其**所在城**；目标须为己方非俘虏武将。
- 校验失败即 no-op：道具不属于该武将、目标为俘虏 等。
- 成功后：道具归属改为该城；该武将（非君主）忠诚 −20（下限 0）。

## 范围 & 不做
- 做：
  - 道具实体与顶层 `items` 集合 + 城/武将道具派生 selector；fixture 播种初始道具。
  - 有效武力/有效智力派生；既有带兵量、开垦/招商公式改吃有效值。
  - 武将忠诚字段（君主派生恒 100）。
  - 赏赐、没收两条**不占人、即时**指令（含 `canX` 校验 + 变更），经 `game.apply/canApply` 分派。
  - 城/武将道具与加成、武将忠诚在 UI 面板的展示与下令入口。
- 不做（本切片）：
  - 忠诚的任何下游规则（叛变、策反、忠诚自然涨跌/回升）。
  - 武将成长/养成；道具的购买/打造/掉落/交易/损坏。
  - 道具加成进入战斗结算的特殊处理（战斗仍只比兵力总和；道具仅经带兵量上限间接影响）。
  - 跨城赏赐/没收；对俘虏的道具操作。

## 验收标准
- [ ] 道具归属唯一存储于 `GameState.items`；城/武将的道具列表均由 selector 派生，无双向冗余。
- [ ] 武将有效武力/智力 = 基础 + 所持道具加成之和；带兵量上限、开垦/招商增量均按有效值计算。
- [ ] 赏赐：合法时道具从城转给同城己方非俘虏武将、忠诚 +8（封顶 100）；武将已持 2 件 / 道具不属于作用城 / 目标为俘虏 时 no-op。
- [ ] 没收：合法时道具从武将收回其所在城、忠诚 −20（下限 0）；道具不属于该武将 / 目标为俘虏 时 no-op。
- [ ] 赏赐/没收**不占人**：下令后目标武将 `busy` 不变，本月仍可被其他指令派遣；二者**立即生效**，不入 `pendingCommands`。
- [ ] 君主忠诚在任何赏赐/没收后仍为 100；普通武将忠诚被钳制在 `[0, 100]`。
- [ ] 玩家入口经 store 注入 `playerLordId` 校验目标武将归属己方；`lordId` 不作为 Action 字段。
